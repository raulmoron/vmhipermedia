API Práctica Hipermedia 2014

La API utiliza los métodos POST, GET y DELETE, para gestionar los datos de la base datos. Exsiten varias llamadas predefinidas para poder gestionar la música y las listas de reproducción, pero la API también permite ejecutar cualquier sql.

Para utilizar los diferentes métodos HTTP y realizar peticiones a la API desde javascript es posible con el uso de XMLHttpRequest. Ejemplo de petición POST:

var xhr = new XMLHttpRequest();
xhr.open("POST","http://hipermedia.local/music/new");
xhr.setRequestHeader('Content-type', 'application/x-www-form-urlencoded');
xhr.send("artist=Artist%20name&title=Title");
xhr.responseText;
...

También con la ayuda de librerías externas, como jQuery ($.post & $.get):

$.post( "http://hipermedia.local/music/new", callbackfunction);

O utilizando el objeto $.ajax:

$.ajax({
  type: "POST",
  url: "http://hipermedia.local/music/new",
  data: "artist=Artist%20name&title=Title",
  success: success,
  dataType: json
});
